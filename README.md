```

Links:             Internal · external · mailto · tel · anchor jumps
Images:            Local · external · responsive · float layout
Tables:            thead · tbody · tr · th · td · caption
CSS Methods:       Inline · internal · external stylesheets
CSS Styling:       Colors · fonts · margins · text-align · float
Google Fonts:      Linking and applying web fonts

FTP:               FileZilla for uploading files to a web server
```

## 🎯 Key Projects

### 🚗 Favorite Links & Dream Cars
First multi-section project combining navigation links and image layout. Features Bentley Motors content with floated images and external links to luxury car brands.

### 🦈 Top 10 Predatory Animals Table
HTML table project ranking predatory animals with images. Demonstrates proper table structure with `thead`, `tbody`, captions, and embedded images.

### 🎨 CSS Methods Comparison
Three-file demo showing the difference between inline, internal, and external CSS — a foundational concept for understanding CSS architecture.

### 🐛 HTML Debugging Lab
Intentionally broken HTML file with 7+ errors to identify and fix — practicing real-world debugging skills.
